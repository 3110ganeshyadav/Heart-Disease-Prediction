# Heart-Disease-Prediction 
# ❤️ Heart Disease Prediction Using Machine Learning

## 📌 Project Overview

This project predicts whether a patient has heart disease using Machine Learning. A Logistic Regression model was trained on a heart disease dataset and evaluated using various performance metrics.

---

## 🚀 Technologies Used

- Python
- Google Colab
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Joblib

---

## 📂 Dataset

Heart Disease Dataset (CSV)

Features include:

- Age
- Sex
- Chest Pain Type
- Resting Blood Pressure
- Cholesterol
- Fasting Blood Sugar
- Resting ECG
- Maximum Heart Rate
- Exercise Induced Angina
- Oldpeak
- Slope
- CA
- Thal

Target:
- 0 = No Heart Disease
- 1 = Heart Disease

---

## 🤖 Machine Learning Algorithm

- Logistic Regression

---

## 📊 Model Performance

| Metric | Score |
|---------|-------|
| Accuracy | 79.5% |
| Precision | 0.76 |
| Recall | 0.87 |
| F1-Score | 0.81 |
| ROC-AUC | 0.88 |

---

## 📈 Project Workflow

1. Import Libraries
2. Load Dataset
3. Explore Dataset
4. Split Features and Target
5. Train-Test Split
6. Train Logistic Regression Model
7. Evaluate Model
8. Generate ROC Curve
9. Save Model
10. Predict on New Patient
